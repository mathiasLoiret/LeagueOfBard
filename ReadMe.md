LEAGUE OF BARD !



Application de visualisation des champions de league of legend.



Le but de l'application est d'utiliser l'API du jeu League Of Legend : 

https://developer.riotgames.com/api/methods

Les informations sont alors affichées dans une CollectionView.


Tous les champions sont présentés et détaillés avec leur nom et leur titre.
